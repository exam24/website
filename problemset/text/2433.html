Introduction

The Raising Bharat Problem Set is a collection of algorithmic programming problems.

Bharat is growing rapidly in Digital & Deep Tech, and it needs strong engineers to lead the way. This course, designed by Rohini Kumar Barla, aims to lay strong fundamentals for students using current industry tools and practices, accelerated learning using Generative AI, understanding the vast opportunities, and fostering an entrepreneurial orientation.

Languages

You can solve problems in several languages, including C, C++, Java, and Python. More information about available languages is here.

Input and output

In all problems you should read input from standard input and write output to standard output.
